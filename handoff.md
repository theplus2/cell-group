# Handoff (현재 상태)

## 📌 현재 상태
- **현재 버전**: v3.0.0
- **빌드 상태**: ✅ 완료 (GitHub Actions Release v3.0 생성됨)
- **주요 URL**: [GitHub Releases](https://github.com/theplus2/cell-group/releases/tag/v3.0)

## 🚀 최근 작업 요약
1. **v3.0 기능 고도화 (v3.0)**:
   - **버전 정보 중앙화**: `src/config.py`로 버전 관리 일원화.
   - **초기화 버튼**: GUI에 데이터 및 설정을 리셋하는 [초기화] 버튼 추가.
   - **나이 제한 강화**: 최소-최대 나이 차이가 10살을 넘으면 **노란색** 경고 및 툴팁 표시.
   - **정렬 기준 명확화**: 조별 멤버 정렬을 `리더 -> 출석점수(성적) 순`으로 고정.
2. **PySide6 마이그레이션 (v2.9)**: Smart App Control 호환성 개선을 위해 PySide6로 전환 완료.

## ⚠️ 알려진 이슈
- Windows Smart App Control이 엄격하게 설정된 경우 여전히 경고가 뜰 수 있음 (코드 서명 미적용 상태).
- 엑셀 저장 시 'openpyxl' 스타일 적용 로직이 데이터 양에 따라 소요 시간이 달라질 수 있음 (최적화 여지 있음).
